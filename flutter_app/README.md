- Home dashboard for clipboard import, quick pick, and settlement entry
- Inventory page for batch/card management
- Pick page for exact-value card extraction
- Settlement page for overview, per-batch accounting, and per-person accounting
- Settings page for role selection and Cloudflare connection config
